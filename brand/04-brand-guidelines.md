### Page 20: Asset Library & Contact

**Asset Library Organization:**

```
/grupoaia-brand-assets/
  /01-logos/
    /primary/          (full logo — script + pillar words)
    /secondary/        (script "Aia" only)
    /icon/             (sweep mark only)
    /monochrome/       (black and white versions)
    /formats/          (SVG, PNG, EPS, PDF for each)
  /02-colors/
    /swatches/         (ASE, ACO, CLR files)
    /palette-reference/ (PDF color guide)
  /03-typography/
    /cormorant-garamond/ (OTF/TTF/WOFF2)
    /inter/              (OTF/TTF/WOFF2)
  /04-templates/
    /business-cards/    (InDesign, PDF)
    /letterhead/        (InDesign, Word, PDF)
    /email-signature/   (HTML)
    /presentations/     (Keynote, PowerPoint, Google Slides)
    /social-media/      (Photoshop, Figma, Canva)
  /05-photography/
    /mood-board/        (reference images)
    /presets/           (Lightroom/Capture One presets)
  /06-illustrations/
    /icons/             (SVG icon set)
    /graphic-elements/  (sweep patterns, dividers)
  /07-guidelines/
    brand-guidelines-v1.pdf
```

**File Format Requirements:**
- Logos: SVG (primary), PNG (with transparency, @1x/@2x/@3x), EPS (print), PDF (print)
- Colors: Adobe ASE, Sketch/Figma palette files
- Templates: Native format + PDF for reference
- Icons: SVG (individual and sprite sheet)
- Fonts: OTF for desktop, WOFF2 for web
